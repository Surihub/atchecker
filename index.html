종이출석부가 없을 때, 빠른 학생 출석체크

사용설명서 보기

1. 명단 입력: 아래 입력칸에 학생 이름을 한 줄에 한 명씩 넣고 '시작' 버튼을 누르세요. 구글시트나 엑셀에 학생의 명단이 있는 경우 학생 이름만 복사해와서 붙여넣는 것을 추천합니다.

2. 출석 체크: 학생의 '미확인' 버튼을 누르면 '출석' → '결석' → '미확인' 순으로 상태가 바뀝니다.

3. 메모: 학생 이름을 클릭하면 메모를 추가하거나 수정할 수 있습니다. (메모가 있으면 📝 아이콘이 표시됩니다.)

4. 정렬: '이름' 또는 '현황' 헤더를 클릭하여 목록을 정렬할 수 있습니다.

5. 복사/초기화: 하단 버튼으로 전체 기록을 복사하거나, 처음부터 다시 시작할 수 있습니다. 서버에 저장되지는 않으므로, 기록을 복사해서 잘 관리해주세요!

※ 주의: 이 버전은 데이터를 저장하지 않습니다. 새로고침하면 모든 내용이 사라집니다.
